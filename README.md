# 🐾 DebugPet

> 코드를 짚어주는 AI 캐릭터 디버깅 도우미

---

## 📌 프로젝트 소개

DebugPet은 코드 오류를 분석하는 AI와,
화면 위에서 직접 오류를 짚어주는 캐릭터를 결합한
인터랙티브 디버깅 시스템입니다.

초보 개발자가 에러를 쉽게 이해할 수 있도록
시각적 안내 + 자연어 설명을 제공합니다.

---

## 🚀 핵심 기능

- 🔍 코드 오류 탐지 (컴파일러 / 린터 기반)
- 🤖 LLM 기반 자연어 설명
- 🐾 캐릭터가 오류 위치 직접 표시
- 🎯 언어별 캐릭터 시스템 (Python, C, Java 등)
- 🖥️ 오버레이 기반 인터랙션 (확장 예정)

---

## 🧱 아키텍처 (단계별)

### A안 - MVP
- 에디터 내부 캐릭터
- 오류 위치 하이라이트
- 기본 설명 제공

### B안 - 오버레이
- 화면 위 캐릭터 이동
- 실제 코드 위치 추적

### C안 - IDE 연동
- VSCode 확장
- 실시간 오류 감지
